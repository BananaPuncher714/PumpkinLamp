# Pumpkin Lamp
Adds custom jack o'lanterns that turn off when a redstone signal is applied to them. Created as a demonstration of [DBL](https://github.com/BananaPuncher714/DimensionalBlocks) and what can be achieved.

## Supported versions
- 1.15.2
